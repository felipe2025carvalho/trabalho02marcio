Cascata

A cascata no CSS é um conceito que define como as regras de estilo são aplicadas aos elementos HTML. O CSS "cascata" funciona com base em uma hierarquia de prioridades e sobreposições para determinar qual regra de estilo será aplicada, caso existam múltiplas regras conflitantes.

Margin

A propriedade margin define o espaço externo de um elemento, ou seja, a distância entre ele e outros elementos ao redor. Diferente do padding, que afasta o conteúdo interno da borda, o margin afasta o próprio elemento de outros elementos.

Especificidade

Especificidade é simplesmente quando navegador usa para decidir qual estilo aplicar quando existem múltiplos estilos que poderiam afetar o mesmo elemento. Quanto maior a especificidade, maior prioridade aquele seletor tem.

Seletores

A propriedade margin no CSS define o espaço externo de um elemento, ou seja, a distância entre ele e os elementos ao redor. Da pra combinar seletores, como:

  • Soma: aplica estilo em dois seletores diferented
  • Espaço: aplica estilo dentro de outro seletor

Pseudo-classes

Elas são usadas para usar em estilos especiais. Pseudo-classes no CSS são palavras-chave adicionadas a seletores que conseguem estilizar elementos com base em seu estado sem precisar de classes adicionais no HTML.

Display Inline e Block

Diferença:

  • Inline: ocupa só o espaço do conteúdo.
  • Block: ocupa a linha a inteira.

Box-Sizing

Com border-box, o tamanho total já inclui o padding e a borda.
